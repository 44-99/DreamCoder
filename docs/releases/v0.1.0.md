# DreamCoder v0.1.0 — local-first AI web game generation reference

> Draft release notes. Do not publish until the maintainer confirms the real prompt-to-preview demo and release tag.

DreamCoder is an open-source, self-hosted reference application that turns natural-language requirements into playable HTML/CSS/JavaScript games, then lets users continue from the existing project files.

## What is included

- A Vue 3 workspace for chat, generated files, and browser preview
- A FastAPI backend with project, authentication, and generation routes
- A LangGraph workflow for requirements, architecture, code, validation, and deployment steps
- DeepSeek, OpenAI, and Qwen-compatible provider configuration
- A Generation Run module that owns project state, transactions, logs, and failure completion
- A Generated Artifact module with traversal, size, entry-file, and atomic-deployment rules
- Local-first SQLite and in-process verification defaults
- Optional PostgreSQL, Redis, ChromaDB, and Docker Compose adapters
- Three offline, single-file game examples for first-use and visual QA
- Backend tests, Ruff linting, frontend production build, and GitHub Actions CI
- Chinese and English README, quickstart, architecture, deployment, and security documentation

## Try the examples without a model key

```bash
python -m http.server 4173
```

Open <http://localhost:4173/examples/> and play Neon Snake, Prism Breakout, or Orbit Dodge.

## Local quickstart

```bash
git clone https://github.com/44-99/DreamCoder.git
cd DreamCoder
cp backend/.env.example backend/.env
```

Add one supported provider key, then follow the [Chinese quickstart](../getting-started.md) or [English quickstart](../getting-started.en.md).

## Important boundaries

- This release primarily targets browser games built with HTML/CSS/JavaScript.
- Generated-code validation is heuristic and is not a full security audit.
- Same-origin preview is not a production multi-tenant sandbox.
- SSE reports workflow steps but is not yet token-level live streaming.
- Provider model IDs and account access can change; verify the official provider catalog.
