# Augment the host agent instead of replacing it

Web2DKit is a Development Quality Layer for coding agents, not a game generator, chat interface, or autonomous agent runtime. The Host Agent continues to own game generation and source changes; Web2DKit owns the workflow and evidence needed to reach a Verified Playable Loop. This boundary avoids duplicating capabilities already provided by Codex, Claude Code, and other hosts while keeping Web2DKit focused on browser-native 2D game quality.
